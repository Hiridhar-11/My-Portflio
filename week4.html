Week 4: Portfolio Development using HTML, CSS & GitHub

1. Overview

This week focused on creating a personal portfolio website from scratch using HTML and CSS. The goal was to design a responsive and visually appealing platform to showcase academic and project credentials, hosted live via GitHub Pages.

2. Key Features

  • Semantic HTML5 layout for clean sectioning
  • Custom CSS for color schemes, fonts, shadows, and transitions
  • Mobile-first responsive design using media queries
  • Top navigation bar with active link highlighting
  • Rounded profile photo, animated typing effect, and text integration

3. Tools & Workflow

  • Visual Studio Code: HTML and CSS development
  • Git: Version control and code tracking
  • GitHub: Remote repository and website hosting
  • GitHub Pages: One-click deployment of static site

4. Enhancements Possible

  • Add JavaScript for scroll animations or modal interactions
  • Use frameworks like Bootstrap or Tailwind for advanced layouts
  • Implement contact form functionality
  • Add interactive project cards with hover effects
  • Include dark/light theme toggle for accessibility

5. Conclusion

By the end of Week 4, a fully functional, responsive portfolio was live on GitHub. This exercise not only enhanced technical web design skills but also introduced professional workflows involving version control and deployment.
