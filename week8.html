Week 8: RFID Access System with Real-Time Google Sheets Logging

1. Introduction

This week, we created an IoT-based RFID door lock system using ESP32, RFID (RC522), buzzer, LED, LCD display, and servo motor, with live logging to Google Sheets.

2. Hardware Setup

  • RC522 RFID Module
  • ESP32 Microcontroller
  • 16x2 I2C LCD Display
  • Buzzer & LED
  • Servo Motor (Gate)
  • RTC Module (DS1302)

3. Functionalities

  • RFID-based user verification
  • LED + buzzer response for access
  • Display user feedback on LCD
  • Servo opens/closes gate for authorized UID
  • Logs UID, status, and timestamp to Google Sheet via Apps Script

4. Code Integration

The code uses libraries for:

  • MFRC522 for RFID
  • LiquidCrystal_I2C for LCD
  • ESP32Servo for gate motor
  • RtcDS1302 for RTC time
  • WiFi and HTTPClient for data logging

6. Firebase & Apps Script Integration

Data from the ESP32 is sent via HTTP POST to a Web App script created using Google Apps Script. The script logs:

  • UID of scanned RFID tag
  • Status (Granted/Denied)
  • Timestamp from RTC

7. Final Output

The system provides real-time, automated attendance or access control that is easily extendable to large scale.
